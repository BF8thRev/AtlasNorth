## Operating Rules (Non-Negotiable)

### 1. Lead with the Problem
Start conversations with margin pressure, delayed feedback, and variability—not Newton features.

### 2. Frame Newton as Visibility, Not Optimization
Newton observes. It doesn't judge. It doesn't fix.

### 3. Emphasize Speed ("Minutes, Not Days")
The key differentiator is feedback speed. When problems are discovered weeks later, it's too late.

### 4. Software First, Hardware Later
Avoid early conversations about installation risk or hardware. Start with logging and visibility.

### 5. Record → Explain → Prevent
This is how trust builds. Don't jump to prevention before explanation.

### 6. Personalize by Role, Not Personality
- COO: margin stability, exposure, confidence
- Director of Extraction: repeatability, attribution
- VP Operations: firefighting, late discovery
- Plant Manager: consistency across shifts

### 7. Email: 50-70 Words
One idea. One question. No scheduling links.

### 8. Information Integrity (MANDATORY)
- NEVER fabricate data
- NEVER assume facts
- NEVER exaggerate capabilities
- NEVER imply unverified results

If it can't be verified, don't use it.

### 9. Forbidden Phrases (NEVER Use)
reaching out, touching base, just wanted to, hope you are well, checking in, following up, excited to, leverage, unlock, game changer, AI powered, cutting edge, world class, optimize, maximize

### 10. CTA = Questions Only
- "Does this show up for you during runs?"
- "Is that gap familiar?"
- "Worth exploring?"

NOT:
- "Would you like to schedule a demo?"
- "Can we set up a call?"

## Integration with CRM

The Newton CRM (Google Sheet) has 3 tabs:

**Tab 1: Outreach Log**
- Tracks every interaction
- Date | Prospect | Company | Message Type | Angle | Channel | Response | Objection Category | Objection Detail | Next Action | Next Action Date

**Tab 2: Company Profile**
- Company-level data
- Company name | Extraction type | Location | Facility size | Pain points | Health/Status | Primary wedge | Source | Notes
- Allows tracking of facility maturity and status across time

**Tab 3: Prospects**
- Individual contacts (allows multiple per company)
- Company name | Contact name | Title | Email | Phone | Status | Last contact | Next action | Next action date | Primary objection | Notes
- Status field: cold / warm / considering / customer
